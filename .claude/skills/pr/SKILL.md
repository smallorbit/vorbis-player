# PR Skill
1. Run `npx tsc --noEmit` and fix any TypeScript errors
2. Run `npm test -- --run` and ensure all tests pass
3. Run `npm run build` and ensure clean build
4. If on main branch, create and checkout a branch for this set of changes.
5. Stage all changes
6. Commit with a descriptive commit message summarizing the changes
7. Push the changes upstream
8. Create a PR of the changes for me to review

Do NOT proceed to the next step if any step fails. Fix the issue first.
